Example Use Cases

Parent Company

This connects multiple customers to each other so we can create views of the data across them

- Warner Bros.

Customer/Division

The group within a Parent company that is requesting the work. The cut-off is at the Finance department and the SOW.

- Warner Music

- Warner Television

Project

A collection of work that is joined thematically for a given customer/business unit

- Warner Oldies Music Videos

- Warner R&B Music Videos

Work Order

A combination of an Input Format (like HD Video) + a process + an output format designed to accomplish part of an overarching project

- Warner Oldies Music Videos (SD)

Workflow

A workflow is the series of steps an asset goes through per direction/payment by customer

- Ingest, digitize and tag an asset with transcription, descriptions and fac rec

Work Step

A logical step in a Work Order that we perform at a cost

- Ingest-to-Digital Master ($40/hour)

- QC ($5 / hour)

- Machine Analyze Face Rec ($13/hour)

Task

The most granular level of activity in the system

- Unbox tapes

- Tag a Video

Task Set

A series of tasks needed to complete a Work Step

Job

A single asset tracked through an end-to-end entire workflow

- Asset 1234BLM tracked through various stages

Logical Assets

Logical assets are the digital media items that are catalogued during a Project. Each item goes through the workflow described in the Work Order.

- A single Music Video
