Tape-to-Data Process

We define what fields to log during the digitization process. This defines the schema for the .CSV export.

 

That "project schema" is an extension of our "base schema"

DMS agrees to log the assets based on our schema and agrees to export that data in a format we define. In the above example, that format is .CSV but it could be that they will deliver it through a web service.

Centaur supports the manually import of this data through the upload of a .CSV file. Even if DMS agrees to the web service, there will be many edge cases where we'll need to do manually import.
